# Projet - Semaine 2 : HTML (Structure & Sémantique)

Ce projet s’inscrit dans la **semaine 2**, dont l’objectif principal est de comprendre et maîtriser la structure fondamentale d’une page web.

Il s’agit ici de la première version de mon **portfolio personnel**, réalisée uniquement en HTML, sans CSS.

## Objectifs de cette étape

Cette phase du projet m’a permis de :

- Comprendre la structure globale d’une page HTML
- Utiliser correctement les balises sémantiques (`header`, `main`, `section`, `footer`)
- Organiser le contenu de manière logique et accessible
- Créer une page de profil personnel complète en HTML pur
- Utiliser des éléments essentiels comme les images, listes, liens et formulaires

## Contenu de cette version (HTML uniquement)

Cette première version du projet comprend :

- Un en-tête avec mon nom et ma photo de profil
- Une section de présentation (biographie)
- Une liste de mes compétences
- Une section dédiée à ma formation
- Des liens vers mes réseaux sociaux
- Un formulaire de contact simple
- Un pied de page

## Suite du projet

Ce projet est une base de travail.  
Dans une prochaine étape (semaine dédiée au CSS), cette page sera **améliorée visuellement** avec du style, une meilleure mise en page et une expérience utilisateur plus agréable.
